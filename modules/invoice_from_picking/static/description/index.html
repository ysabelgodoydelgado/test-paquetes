Odoo customer Invoice From Picking(Delivery Order/ Incoming Shipment)

In Odoo with this app create customer invoice from picking (Delivery Order/ Incoming Shipment).for warehouse, accounting and invoicing this app easy way around many user, who are managing different vendor and their stock inventory dependent on supplier this app suits for that even it automate that process in Odoo. with this app you can create vendor bill and purchase invoice too for supplier. Invoice from Picking (Deliveries To Invoice and Incoming Shipment) feature is only available in odoo version 8, After odoo v8 this feature is deprecated. So we create custom module to add this functionality.

Features

Users are allowed to create invoices

Users are allowed to create invoices from done delivery orders.

User can select invoice/bill Date

Users are allowed to select the invoice date from the wizard.

Users are allowed to create purchase invoices

Users are allowed to create purchase invoices from delivery order.

Users can select different invoice type.

Users are allowed to select different invoice types from the wizard.

Sale order view

You can see sale order view.

Deliveries To Invoice Menu

You can see deliveries to invoice from Sales > Invoicing > Deliveries to Invoice menu.

Create Draft Invoice.

After confirming delivery order you can create draft invoice by clicking create invoice button.

You can also create invoice from action menu.

Create Invoice Wizard.

When you click on create invoice button, One wizard will be open. In that wizard you can select invoice date and also apply group by filter in partner.

View Invoice.

After creating invoice from picking you can see created invoice in tree view.

Delivery invoice form view.

Created invoice form view.

Here you can see created invoice linked with picking (Delivery Order).

Changed Delivered Quantity

Delivered quantity and invoiced quantity will be changed.
